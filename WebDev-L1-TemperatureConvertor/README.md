# Task 3-Temperature Convertor

## Description 
A simple web application that converts temperatures between celsius, farenheit and kelvin.

## Features
- Convert Celsius into Farenheit and Kelvin.
- Convert Farenheit into Celsius and Kelvin.
- Convert Kelvin into Farenheit and Celsius.
- Responsive Design.
- Input Validation.
- Clean and modern UI.

## Technologies used
- HTML
- CSS
- Javascript

## How to run the project
1-Download or clone the repository
2-Open the project folder
3-Double click on "WarishaMateen_Task3.html" file(or open it with any browser).
4-The temperature convertor will open and is ready to use
